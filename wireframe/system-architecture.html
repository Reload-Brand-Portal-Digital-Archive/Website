3. Request Flow & Middleware Chain

Alur sebuah HTTP request dari browser hingga database

🌐 User Browser
axios.get/post/put/delete
HTTP Request + Authorization: Bearer <JWT>
⚡ Express Server (Port 5000)
cors() → express.json() → express.urlencoded()
Route matching /api/*
🛡️ Middleware Layer
verifyToken → isAdmin → uploadMiddleware
req enriched with user + file
📦 Controller Function
Business logic + SQL query builder
pool.execute(sql, params)
🗄️ MySQL (reload_db)
Connection Pool → Query → Result
res.json({ data })
💻 React Component
setState / useState → re-render UI
🔐 JWT Authentication Flow
POST /api/auth/login
→
bcrypt.compare()
→
jwt.sign(payload, SECRET)
→
{ token }
Token disimpan di localStorage → dikirim via header Authorization: Bearer <token>
🛡️ Middleware Guard Levels
LevelMiddlewareEfek jika gagal
Public—Akses bebas
OptionaloptionalVerifyTokenGuest tetap dilanjutkan
UserverifyToken401 Token tidak ditemukan
AdminverifyToken + isAdmin403 Bukan admin
📊 Tracking System Flow
1. RouteTracker di App.jsx listen setiap perubahan lokasi 2. trackPageView(pathname) dipanggil otomatis 3. Cookie client_id di-generate/retrieve 4. POST /api/track/pageview dengan { url, client_id } 5. Click Shopee/TikTok → POST /api/track/click/:platform 6. Backend redirect URL dikembalikan → window.open(url) 7. Admin melihat stats via GET /api/track/stats
